API Documentation

URL de base pour les endpoints : https://mines-io.herokuapp.com


Endpoints Description
GET /leaderboard Renvoie des informations sur les 5 premiers joueurs
GET /users Renvoie la liste des joueurs
GET /users/:name Renvoie les informations sur un joueur
POST /users Créé un joueur et renvoie la liste des joueurs
PUT /users/:name Modifie les informations d'un joueur et les renvoie
DELETE /users/:name Supprime un joueur et renvoie la liste des joueurs

  • GET /leaderboard

                        "username": "marceau",
                        "size": 41
                    }, {
                        "username": "julien",
                        "size": 12
                    }]
                }

  • GET /users/:name

Renvoie les informations sur un joueur.

                {

                    "size": 150,

  • POST /users

Créé un joueur et renvoie la liste des joueurs. Les informations du joueur sont à renseigner dans le body de la requête HTTP.

Ne pas oublier de renseigner dans l'entête de la requête : Content-Type: application/json

Exemple de body : 
            
                {
                    "ciarletta": {
                        "position": [
                            40,
                            56
                        ],
                        "size": 436,
                        "color": "vaxou"
                    }
                }
Exemple de réponse :
            
                {
                    "vaxou": {
                        "position": [
                            42,
                            7
                        ],
                        "size": 76,
                        "color": "vaxou"
                    },
                    "todor": {
                        "position": [
                            70,
                            70
                        ],
                        "size": 150,
                        "color": "vaxou"
                    },
                    "ciarletta": {
                        "position": [
                            40,
                            56
                        ],
                        "size": 436,
                        "color": "vaxou"
                    }
                }

  • PUT /users/:name

Modifie les informations d'un joueur et les renvoie. Les informations du joueur sont à renseigner dans le body de la requête HTTP.

Ne pas oublier de renseigner dans l'entête de la requête : Content-Type: application/json

Exemple de requête :

PUT https://mines-io.herokuapp.com/users/ciarletta

Exemple de body :
            
                {
                    "position": [
                        100,
                        100
                    ],
                    "size": 3000,
                    "color": "vaxou"
                }
Exemple de réponse :
            
                {
                    "position": [
                        100,
                        100
                    ],
                    "size": 3000,
                    "color": "vaxou"
                }
